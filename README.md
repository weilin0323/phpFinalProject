## 教務處課程查詢系統

是個可供同學查詢各種課程資訊及下載考古題的教務系統，系統首頁可了解各教授的課程資訊、評價等，此外，還可以上傳歷屆考古幫助同學，解決同學選課及沒有考古練習的問題。<br/>
